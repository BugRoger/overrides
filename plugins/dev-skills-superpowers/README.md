# dev-skills-superpowers

Superpowers-based development skills - enhanced workflow patterns for Claude Code.

## Installation

```bash
/plugin marketplace add bugroger/overrides-marketplace
/plugin install dev-skills-superpowers@overrides-marketplace
```

## Skills

- **brainstorming** - Structured brainstorming sessions
- **dispatching-parallel-agents** - Run multiple agents in parallel
- **executing-plans** - Execute implementation plans step by step
- **finishing-a-development-branch** - Complete and prepare branches for merge
- **parallel-batch-execution** - Batch execution of parallel tasks
- **receiving-code-review** - Handle and address code review feedback
- **requesting-code-review** - Request thorough code reviews
- **subagent-driven-development** - Development using subagents
- **systematic-debugging** - Methodical debugging approach
- **test-driven-development** - TDD workflow patterns
- **using-git-worktrees** - Git worktree management
- **using-superpowers** - Guide to using superpowers skills
- **verification-before-completion** - Verify work before marking complete
- **writing-plans** - Write detailed implementation plans

## Agents

- **plan-executor** - Execute plans with verification
